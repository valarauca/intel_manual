2.2
IA-32E MODE
IA-32e mode has two sub-modes. These are:
•
Compatibility Mode.
Enables a 64-bit operating system to run most legacy protected mode software
unmodified.
•
64-Bit Mode.
Enables a 64-bit operating system to run applications written to access 64-bit address space.
